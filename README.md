 Making a Website Mobile-Friendly Using CSS Media Queries
This project shows how to convert a desktop-only webpage into a fully responsive, mobile-friendly website using CSS media queries. The goal is to ensure that the layout adjusts smoothly on phones, tablets, and desktop screens.

 Objective
Make an existing website mobile-friendly

Apply CSS media queries for different screen sizes

Improve layout, readability, and user experience on small devices

Ensure images, text, and sections scale correctly

 Tools Used
HTML5

CSS3

Media Queries

VS Code

Chrome DevTools (for mobile testing)

 What Was Done
Added the responsive <meta> viewport tag

Converted fixed widths into flexible layouts

Made navigation mobile-friendly

Stacked two-column layouts into a single column on small screens

Adjusted text size, spacing, and images

Used media queries to apply styles only on smaller devices

 Responsive Breakpoints Used
Device	Media Query
Mobile	@media (max-width: 600px)
Tablet	@media (max-width: 900px)
Desktop	Default (no media query)
Files Included

Testing Responsiveness
To test mobile view in Chrome:

Open the website

Press F12

Click the device toolbar icon (phone/tablet)

Select a device like iPhone 12

Adjust CSS until the layout fits perfectly

 Outcome
The final website:

Looks good on all screens

Has readable text on mobile

Has auto-scaling images

Has sections that stack neatly

Provides better user experience

 Conclusion
CSS media queries are the simplest way to make any website mobile-friendly.
With a few breakpoints and layout adjustments, even older fixed-width pages can become responsive.
